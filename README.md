# ✈️ Real-Time Flight Data Abnormalities Detection

An AI-powered system that detects anomalies in real-time from aircraft sensor logs to enhance flight safety. The system leverages deep learning and statistical methods to identify unusual flight behavior and enable proactive responses to potential failures.

---

## 📌 Project Overview

Modern aircraft generate massive volumes of sensor data during flight. Detecting anomalies in these logs is crucial for early fault detection, predictive maintenance, and improving aviation safety.
This project processes real-time flight data, applies preprocessing techniques, and uses anomaly detection algorithms to flag abnormal patterns that could indicate risks or system faults.

---

## 🧠 Key Features

- ✅ Real-time ingestion and processing of flight sensor data
- 📉 Anomaly detection using deep learning and statistical models
- 📊 Visual dashboards to display and compare normal vs abnormal flight patterns
- 🚨 Early warning system for aviation safety monitoring
